# Surfing App - Flutter UI

一个使用Flutter开发的现代化冲浪应用UI界面，展示了优雅的设计和流畅的用户体验。

## 项目特点

- 🏄‍♂️ 现代化的冲浪社交界面设计
- 📱 响应式布局，支持多种设备尺寸
- 🎨 精美的UI组件和动画效果
- 🖼️ 故事流和个人资料展示功能
- 💫 流畅的页面过渡效果

## 技术栈

- Flutter SDK
- Dart语言
- SVG图标支持
- 自定义Widget组件

## 开始使用

### 环境要求

- Flutter (最新版本)
- Dart SDK
- Android Studio / Trae
- iOS模拟器 / Android模拟器

### 安装步骤

1. 克隆项目到本地：
```bash
git clone https://gitcode.com/nutpi/flutter_Surfing_App.git
```

2. 进入项目目录：
```bash
cd flutter_Surfing_App
```

3. 安装依赖：
```bash
flutter pub get
```

4. 运行项目：
```bash
flutter run
```

## 项目结构

```
lib/
├── Models/          # 数据模型
├── Screens/         # 页面UI
│   ├── FollowScreen/    # 关注页面
│   ├── LandingScreen/   # 着陆页
│   └── ProfileScreen/   # 个人资料页面
├── Widgets/        # 可复用组件
├── constants.dart  # 常量定义
└── main.dart       # 应用入口

assets/
├── icons/         # SVG图标
└── images/        # 图片资源
```
